# beardiest.com

Personal website and portfolio for beardiest.

This repository contains a plain HTML, CSS, and JavaScript site used to present projects, experiments, and contact information. The site is intentionally framework-free and kept lightweight so it can be deployed as a static website.

## Overview

- Landing page with hero, about, contact, and projects sections
- Dedicated project detail pages under the `projects/` directory
- Custom visual styling, interactive project graph, and lightweight client-side effects
- Static hosting friendly setup for GitHub Pages or any simple web host

## Tech Stack

- HTML
- CSS
- JavaScript
